A WordPress plugin that remove all default content created by Wordpress.

== Description ==
This plugin if for use only once in each WordPress installation becouse it removes the following default conent:

   * Default "Hello World" post.
   * The "Example page".

Also changes the name and slug of the default category created by WordPress to "News".



== Installation ==

1. Visit your admin section and click Plugins -> Add New.  Then search for "WP Remove Default Content".

2. After activate this plugin you can review the default content.

3. The best option is to deactivate and uninstall this plugin.
